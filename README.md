# LightVPS

> [!CAUTION]
> ### Скрипт прошел бета тест, возможность ошибок минимальная

---

**LightVPS** — это простой Bash-скрипт для первичной настройки нового сервера.

**Работает на:** Ubuntu 24-25-26 и Debian 12

<img width="291" height="330" alt="изображение" src="https://github.com/user-attachments/assets/8abe11b2-ead3-43aa-9ff1-d5f7c3dd3f28" />

Он сделан для тех, кто не хочет каждый раз вспоминать десятки команд вручную и боится что-то забыть при настройке VPS. Скрипт помогает пройти базовые шаги по порядку: подготовить систему, создать пользователя, настроить SSH, включить защиту, поставить Docker и проверить текущее состояние сервера.

## Что умеет скрипт

Скрипт умеет:

- обновлять систему и устанавливать полезные пакеты;
- показывать информацию о сервере;
- создавать новых пользователей;
- добавлять SSH-ключи;
- настраивать SSH;
- включать UFW и Fail2Ban;
- настраивать DNS;
- менять timezone;
- включать BBR;
- устанавливать Docker;
- выполнять откат некоторых настроек;
- очищать ноду от Docker-контейнеров, временных файлов и прочего мусора.

## Что внутри удобно сделано

- текстовое описание каждого пункта меню при его выборе
- обычное текстовое меню по цифрам;
- русский язык;
- есть `dry-run` для безопасной проверки без применения изменений;
- есть `self-test` для быстрой самопроверки скрипта.

## Кому подойдёт

Скрипт подойдёт:

- новичкам, которые только начинают работать с VPS;
- тем, кто поднимает серверы не каждый день и не хочет вспоминать всё с нуля;
- тем, кто хочет держать базовую настройку в одном месте.

## Кому не подойдёт

Скрипт может не подойти, если:

- вы хотите очень узкую кастомную настройку под нестандартную инфраструктуру;
- у вас уже сильно настроенный сервер, где опасно что-то менять общим скриптом;
- вы предпочитаете всё делать вручную и точно знаете, что делаете.

## Как запускать

Автоматическое скачивание с Гитхаб в папку `/root`, статус "исполняемый", запуск файла

```bash
cd /root && \
( command -v curl >/dev/null 2>&1 && curl -fsSL -o /root/vps.sh https://raw.githubusercontent.com/OMchik33/LightVPS/refs/heads/main/vps.sh || \
  command -v wget >/dev/null 2>&1 && wget -O /root/vps.sh https://raw.githubusercontent.com/OMchik33/LightVPS/refs/heads/main/vps.sh ) && \
chmod +x /root/vps.sh && \
/root/vps.sh
```

Самопроверка:

```bash
bash vps.sh --self-test
```

Безопасный пробный запуск без применения изменений:

```bash
bash vps.sh --dry-run
```

Обычный запуск:

```bash
bash vps.sh
```

## Важный совет перед применением SSH-настроек

Если скрипт меняет SSH-порт, отключает вход по паролю или меняет доступ для `root`, **не закрывайте текущую сессию сразу**.

Сначала всегда проверьте вход в **новом окне терминала**. Только после этого можно считать настройку успешной.

## Главное о подходе

Этот скрипт не заменяет понимание сервера, но сильно упрощает старт. Его задача — снять рутину, а не лишить вас контроля.

Если использовать его спокойно и по шагам, он помогает сделать первичную настройку VPS быстрее, понятнее и аккуратнее.
